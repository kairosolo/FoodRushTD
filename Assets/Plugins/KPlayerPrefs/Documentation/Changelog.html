Features

  • Global Data: Shared settings across all profiles (graphics, audio, preferences)
  • Profile Data: Player-specific progress and game state
  • Direct Access: Access any profile without switching active profile
  • Event System: Notifications for data changes and profile operations
  • File Operations: Export/import data for backups and migration
  • Editor Tools: Complete data management from Unity editor
